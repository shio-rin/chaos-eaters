RULES

  • Keep the formatting consistent! When using an italics font, set the color to the 3rd leftmost top color (there’s probably a specific name for this on pc), use size 12 for normal text, 20 for a heading/title, 17 for a subtitle, and 14 for a sub-subtitle. Subtitles need to be all caps and bold. always underline subtitles and titles. The whole doc uses the “Nunito” font (other than some of the titles, which use Oswald. Most of it uses 1.15 spacing, although titles can be different sometimes (ex. if you want to add a subtitles close or further away from the title without indents) Otherwise just use the drafting area honestly
  • If you want to add somebody else as an editor, make a Google Form and paste the link into the john editors page (people with editor will be automatically added). We need at least ⅓ the editors to vote (3 people at the time of writing), and if ¾ of voters say yes, (at the time this is 3 people, but if ex. 3 people vote yes but 3 vote no, the person is not added because only 50% of those who voted [6] voted yes), you may add that person into the doc as editor.
  • You may add as many people as viewers or commenters, however if you want somebody to view, the John monologue doc can be viewed by anyone with a link. If you find any of the commenters annoying (ex. they are adding to many nonsense or unrelated suggestions) feel free to switch them to viewer, but do not remove them unless they are banned.
